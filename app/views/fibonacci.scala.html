GET /api/v1/fibonacci/:method/:request

Return single or sequence of Fibonacci numbers which is calculated with one of the available calculation method.

Resource URL

http://localhost:9000/api/v1/:method/:request

Parameters

  • method

    Fibonacci calculation method
    required
    Available methods:
    • - recursive
    • - dynamic
    • - bine

  • request

    Calculation request
    required
    Examples:
    • 5 - get 5th Fibonacci number
    • 1-10 - get sequence of Fibonacci numbers start from 1th to 10th
    • 1,3,5-15 - complex request

Example Request

http://localhost:9000/api/v1/fibonacci/recursive/1-5,10

Example Result

[[0,1,1,2,3],34]
